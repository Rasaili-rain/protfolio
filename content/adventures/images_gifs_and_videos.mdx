# Making My Blog Visual

Text is powerful—but visuals make content **engaging**.  
In this post, I’m showcasing how I use **images, GIFs, and media** inside my Next.js MDX blog.

---

## Using Static Images

For normal images, I place files inside:
public/next.svg

Then reference them directly in MDX:
![Next.js Blog Setup](/next.svg)
Fast  
SEO-friendly  
Works perfectly with Next.js static optimization



## Using Next.js `<Image />` Component
For better performance and optimization:

```tsx
import Image from "next/image";

<Image
  src="https://picsum.photos/seed/picsum/200/300"
  alt="Unsplash demo"
  width={200}
  height={300}
/>
````
